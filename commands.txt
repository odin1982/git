git --version                               obtiene la version de GIT
git init                                    inicializa un repositorio en un directorio existente
git clone <url>                             clona el repositorio de la <url> especificada
git clone <url> <new-name-repository>       clona el reposito de la <url> y lo renombra <new-name-repository>
git status                                  checa el estatus de tus archivos
